# tinycubesat.github.io

Website

## Env

### Create env

#### Linux

    virtualenv .venv

#### Windows

    python -m venv .venv

### Activate env

#### Linux

    source .venv/bin/activate

#### Windows

    .venv/Scripts/activate

## Install requirements

    pip install mkdocs-material

## Run

    mkdocs serve

## Building

    mkdocs build
